Overview

Download raw dataset from Andersson et al.Â 2021

HER2 breast cancer (Spatial Transcriptomics platform) dataset can be accessed at https://doi.org/10.5281/zenodo.4751624. After downloading and unzip, the following files will be needed:

  • the gene expression matrix

  • Coordinates

  • raw H&E image (TIFF)

tstamp <- gsub("-", "", as.character(Sys.Date()))
data_path <- "/data/cqs/ywang/source/analysis"

positionTableFile <- file.path(data_path, "H1_position.csv")
positionTable <- read.csv(positionTableFile, header = F, row.names = 1)
positionTable <- positionTable[, c(4, 5)]
colnames(positionTable) <- c("row", "col")

countTableFile <- file.path(data_path, "H1.tsv")
countTable <- read.table(countTableFile, header = T, row.names = 1, sep = "\t")
row.names(countTable) <- paste0("X", row.names(countTable))

rawimg <- file.path(data_path, "HE_BT24044_D2.jpg")

sampleName <- "HER2.H1"
patchsize <- 256


Extract features

Image features and RGB quantile matrix will be extracted with VGG16 model of ImageNet from the raw H&E image.

imageFeatures_list <- ExtractFeatures(imgFile = rawimg, positionTable = positionTable, patchSize = patchsize,
    rawImage = T, savePatchOnImage = paste0(sampleName, "_rawpatch_", patchsize, ".png"))

imageFeatures <- imageFeatures_list[["ImageFeature"]]
RGBquantile <- imageFeatures_list[["RGBquantile"]]

Or they can be extracted with ResNet50 model using pytorch. And the extracted matrices can be directly loaded.

HER2_Obj <- LoadImageFeature(countTable = t(countTable.filter), imageFeatures = imageFeatures, positionTable = positionTable,
    RGBquantile = RGBquantile, project = sampleName)
